# 등록 준비 체크리스트

## 사전확정

- [ ] 사업자 형태 결정
- [ ] 영업범위 결정
- [ ] 상호 후보 확인
- [ ] 대표자·임원·업무총괄사용인 결격사유 확인
- [ ] 영업소 관할 구·군 결정

## 물적 요건

- [ ] 최신 자기자본·순자산 기준 확인 및 증빙
- [ ] 고정사업장 적격성 사전확인
- [ ] 임대차계약서 또는 소유권 증빙
- [ ] 전화·통신·전산설비
- [ ] 개인정보 문서 잠금·접근통제 설비
- [ ] 간판·전화번호·영업소 정보 확정

## 교육·보장

- [ ] 대표자 또는 업무총괄사용인 등록교육
- [ ] 교육이수증 유효기간 확인
- [ ] 보증금 예탁·보험·공제 중 방식 선택
- [ ] 손해배상책임 보장 증서 준비

## 신청서류

- [ ] 최신 등록신청서
- [ ] 교육이수증
- [ ] 영업소 증빙
- [ ] 자본·순자산 증빙
- [ ] 결격사유 관련 서류
- [ ] 손해배상책임 보장 증서
- [ ] 법인 등기·정관·주주 관련 서류(법인인 경우)
- [ ] 담당부서 요구 추가서류

## 등록 후

- [ ] 등록증 기재사항 검토
- [ ] 사업자등록 또는 업종 정정
- [ ] 제휴 대부업체 등록 유효성 확인
- [ ] 서면 제휴계약
- [ ] 고객서식·상담대장·민원대장 마련
- [ ] 광고 사전검수
